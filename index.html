Jobseek

Built using: React, PostgreSQL, Express, Node.js, react-beautiful-dnd, CSS

Keep track of your job applications on this kanban-inspired dashboard.

With Jobseek, users can create and delete cards that represent job applications.

Each card keeps track of some basic details tied to each application.

The drag and drop feature serves to provide a smooth and fun user experience!

Visions

Built using: React, PostgreSQL, Express, Node.js, EmailJS, CSS

Visions is an ecommerce website that allows you to browse professional photos and place and order for them.

Users can add and remove items from cart, change the quantity of an item in cart, and place and order for those items.

Upon placing an order, an email will be sent to the user with the order details.

Boba Finder App

Built using: HTML, CSS, JavaScript, jQuery

Wondering where you can find boba in the city that you are in?

This app asks you for the name of a city and makes an API call to the Google Maps Places API.

It will then return a list of boba shops in that city

along with it's Google rating out of 5 and address.
